# 📖 Contributing Guidelines

Thank you for your interest in this project! Please read these rules carefully before creating an issue or attempting 
to contribute.

---

## 🚫 Code contributions (Pull Requests)

- Development of this application is carried out **exclusively by members of the organization**.  
- **All external pull requests will be automatically closed** - we do not accept code contributions from outside the organization.  
- If you wish to suggest a change, please open a **Feature request** or **Bug report** issue instead (see below).  

---

## 🐞 Reporting bugs & ✨ Suggesting features

- Users are welcome to open:
    - **Bug reports** (for problems or malfunctions in the application)
    - **Feature requests** (for new ideas or enhancements)

- **Issues of any other category will be automatically closed.**  
- Please use the **provided issue forms** when submitting bugs or feature requests. They help us gather all the necessary information quickly and efficiently.

---

## 📌 Summary

✅ **Allowed:**  
- Bug reports  
- Feature requests  

❌ **Not allowed:**  
- Pull requests from external contributors  
- Issues that don’t fall into one of the above two categories  

Thank you for respecting these rules - this helps us keep the project clean, structured and easy to maintain.
